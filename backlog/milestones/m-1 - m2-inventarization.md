## Description

System180's inventory/asset management app built on top of the core foundation. Runs on the Jetson with 3 cameras. Features: live 3-camera view with bounding box overlays, part identification via YOLO, ArUco-based mm measurement, ROI calibration, side cameras triggered on demand (ambiguous classes only), and a clean web UI for System180 operators. This milestone has the highest priority and must be completed before Gamification work begins.
